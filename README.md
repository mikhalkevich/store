## Laravel Store
Laravel 
- git clone http://github.com/mikhalkevich/store.git
- создать базу store
- создать файл .env (для настроек Laravel)
- в файле .env подключаемя к базе данных store
- composer self-update
- composer install
- php artisan migrate
